# ⚠️ عام مسائل کا حل
## Troubleshooting

</div>

<details>
<summary><h3>❌ ModuleNotFoundError</h3></summary>

<div align="center">

```diff
- Error: ModuleNotFoundError: No module named 'xxx'
```

</div>

**حل:**
چیک کریں کہ `requirements.txt` انسٹال کی ہے:

```bash
pip install -r requirements.txt
```

</details>

<details>
<summary><h3>❌ ماڈل ایرر | Model Error</h3></summary>

<div align="center">

```diff
- Error: Model not found / Failed to load model
```

</div>

**حل:**
`Models/` فولڈر چیک کریں کہ فائلز صحیح جگہ رکھی ہیں۔

</details>

<details>
<summary><h3>❌ GPU نہیں چل رہا</h3></summary>

<div align="center">

```diff
- Warning: CUDA not available, using CPU
```

</div>

**حل:**
1. پرانی torch فائلز ڈیلیٹ کریں
2. دوبارہ GPU والی کمانڈ چلائیں

```bash
pip uninstall torch torchvision torchaudio
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```

</details>
