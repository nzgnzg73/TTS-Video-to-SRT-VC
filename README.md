## ğŸŒ API Documentation
The server provides REST API endpoints:


## TTS Generation
```bash
POST /tts
Content-Type: application/json

{
  "text": "Your text here",
  "voice_mode": "predefined",
  "predefined_voice_id": "voice1",
  "temperature": 0.8,
  "speed_factor": 1.0,
  "language": "en",
  "output_format": "mp3"
}
```


## Voice Cloning

```bash

POST /process_vc
Content-Type: multipart/form-data

source_audio: [file]
target_audio: [file]
device: "cuda"
cfg_rate: 0.5
sigma_min: 0.000001
```
## Transcription



```bash
POST /transcribe
Content-Type: multipart/form-data

file: [audio/video file]
language: "en"
model: "small"
task: "transcribe"
```
